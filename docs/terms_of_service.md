# Пользовательское соглашение

**Сервис: Сразуум**  
**Дата последнего обновления: 04.01.2026**

**Контакт для связи:** @vee_support (Telegram)

## 1. Общие положения

Настоящее Пользовательское соглашение (далее — «Соглашение») регулирует отношения между администрацией сервиса кросспостинга между Telegram и MAX «Сразуум» (далее — «Сервис») и пользователем Сервиса (далее — «Пользователь»).

Используя Сервис, Пользователь соглашается с условиями настоящего Соглашения.

## 2. Предмет соглашения

Сервис предоставляет Пользователю возможность настройки автоматического кросспостинга контента между каналами в Telegram и MAX.

## 3. Регистрация и использование Сервиса

3.1. Для использования Сервиса Пользователь должен:
- Иметь аккаунт в Telegram
- Иметь доступ к каналам в Telegram и MAX
- Добавить бота в качестве администратора в соответствующие каналы

3.2. Пользователь несет ответственность за:
- Достоверность предоставленной информации
- Безопасность доступа к своему аккаунту
- Соблюдение прав третьих лиц при использовании Сервиса

## 4. Условия использования

4.1. Пользователь обязуется:
- Использовать Сервис только в законных целях
- Не нарушать права третьих лиц
- Не использовать Сервис для распространения незаконного контента
- Соблюдать правила использования платформ Telegram и MAX

4.2. Запрещается:
- Использовать Сервис для распространения спама, вредоносного контента или материалов, нарушающих законодательство
- Предпринимать попытки обхода ограничений Сервиса
- Передавать доступ к аккаунту третьим лицам без согласия администрации

## 5. Подписки и платежи

5.1. Сервис предоставляет:
- Бесплатный период для первой связи (30 дней)
- Платные подписки для дополнительных связей
- Специальные условия для VIP-пользователей

5.2. Оплата подписки осуществляется через платежную систему YooKassa.

5.3. Все платежи являются невозвратными, за исключением случаев, предусмотренных законодательством.

5.4. Администрация оставляет за собой право изменять тарифы, предварительно уведомив Пользователей.

## 6. Ответственность

6.1. Сервис предоставляется «как есть». Администрация не гарантирует бесперебойную работу Сервиса и не несет ответственности за:
- Временную недоступность Сервиса
- Потерю данных по независящим от администрации причинам
- Действия третьих лиц

6.2. Пользователь несет полную ответственность за контент, который публикуется через Сервис.

6.3. Администрация не несет ответственности за нарушения, совершенные Пользователем на платформах Telegram и MAX.

## 7. Интеллектуальная собственность

Все права на Сервис, включая программное обеспечение, дизайн и контент, принадлежат администрации или соответствующим правообладателям.

## 8. Прекращение доступа

Администрация вправе прекратить или ограничить доступ Пользователя к Сервису в случае:
- Нарушения условий настоящего Соглашения
- Распространения незаконного контента
- Других действий, нарушающих законодательство или права третьих лиц

## 9. Изменения в Соглашении

Администрация оставляет за собой право вносить изменения в настоящее Соглашение. Актуальная версия всегда доступна в Сервисе. Продолжение использования Сервиса после внесения изменений означает согласие Пользователя с новыми условиями.

## 10. Контакты

По всем вопросам, связанным с использованием Сервиса, вы можете обратиться через Telegram бота.

## 11. Применимое право

Настоящее Соглашение регулируется законодательством Российской Федерации.
